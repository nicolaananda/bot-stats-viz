STOCK MANAGEMENT API CONTRACT - UPDATED
==========================================

BASE URL: https://api-botwa.nicola.id/api/dashboard

ENDPOINTS YANG PERLU DITAMBAHKAN:
=================================

1. ADD STOCK
   POST /products/{productId}/stock/add
   
2. GET STOCK ITEM  
   GET /products/{productId}/stock/item/{stockIndex}
   
3. EDIT STOCK ITEM
   PUT /products/{productId}/stock/item/{stockIndex}
   
4. DELETE STOCK ITEM
   DELETE /products/{productId}/stock/item/{stockIndex}
   
5. DELETE MULTIPLE STOCK
   DELETE /products/{productId}/stock

NOTES:
======
- Endpoint GET /products/{productId}/stock/details sudah ada dan berfungsi
- Yang perlu ditambahkan adalah 5 endpoint di atas dengan format /stock/item/ untuk individual operations
- Format request/response sama seperti dokumentasi STOCK_API_DOCUMENTATION.md
- CORS harus diaktifkan untuk frontend
